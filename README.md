# Blockchain Staking DApp Scrum Project

## Overview

This repository demonstrates the application of Scrum principles in the delivery of a decentralized staking platform.

The project was managed using Scrum and showcases Agile planning, backlog refinement, sprint execution, reviews, retrospectives, and Jira reporting.

---

## My Role

**Scrum Master**

### Responsibilities

* Facilitated Sprint Planning
* Managed Product Backlog
* Supported backlog refinement
* Facilitated Sprint Reviews
* Facilitated Sprint Retrospectives
* Tracked sprint progress in Jira

---

## Product Goal

Develop a staking application that enables users to:

* Connect wallets
* Stake tokens
* View staking balances
* Claim rewards
* Unstake assets
